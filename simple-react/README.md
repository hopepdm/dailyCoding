# simple-react

一个极简的React框架，用于说明React实现原理，请不要用于生产环境

实现了JSX，组件，生命周期，diff等核心功能

## 运行
```bash
git clone https://github.com/hujiulong/simple-react.git
cd simple-react
npm install
parcel index.html
```

说明：使用diff算法可以优化渲染效率，不用重新渲染全局dom，因为dom操作非常消耗性能，因此使用diff算法可以做到渲染变化的dom。
